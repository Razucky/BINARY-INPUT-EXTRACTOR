# Binary Input Extractor

Extract binary inputs from protection switchboard PDFs with a simple GUI.

## üöÄ Quick Start - Get the Executable (No Python Required!)

### Step 1: Download the latest release
Go to the **Releases** section (right sidebar) and download:
- **Windows:** `BinaryInputExtractor.exe`
- **Mac:** `BinaryInputExtractor` 
- **Linux:** `BinaryInputExtractor`

### Step 2: Run it
- **Windows:** Double-click the `.exe` file
- **Mac:** Right-click ‚Üí Open (first time only, to bypass Gatekeeper)
- **Linux:** `chmod +x BinaryInputExtractor && ./BinaryInputExtractor`

---

## üìã How to Use

1. Click **Browse** to select up to 3 PDF files
2. Choose where to save the output Excel file
3. Click **Extract Binary Inputs**
4. Each PDF becomes a separate tab in the Excel file

---

## üîß Building Your Own Executables

If you want to build fresh executables:

1. Fork this repository
2. Go to **Actions** tab
3. Click **Build Executables**
4. Click **Run workflow**
5. Wait ~5 minutes
6. Download from **Releases** or **Artifacts**
